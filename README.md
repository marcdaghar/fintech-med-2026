## 📖 Description

Ce projet implémente un **système financier alternatif** pour la coopération méditerranéenne, combinant trois piliers mathématiques :

1. **Thermodynamique économique** : Analyse des fuites de valeur via le flux entropique
2. **Transport optimal** : Optimisation logistique par l'algorithme de Sinkhorn
3. **Waqf numérique** : Redistribution équitable via contrat intelligent
